# User Manual / User Guide - ASPO Larkana System

## 1. Getting Started
### 1.1 Accessing the System
Navigate to the application URL (e.g., `http://localhost:3000`).
The login screen will appear. Enter your username and password provided by the system administrator.

### 1.2 Language Selection
The system supports English and Urdu.
- **Top Bar**: Click on the language icon (flag/globe) to toggle between languages.
- **RTL Support**: When Urdu is selected, the layout will automatically reverse to support Right-to-Left reading.

## 2. Global Navigation
- **Sidebar**: Access all modules (Staff, Offices, Leave, Revenue, etc.).
- **Top Bar**: Home, Language Toggle, and Log Out.
- **Bottom Nav (Mobile)**: Primary actions available at a glance on mobile devices.

## 3. Core Module Workflows

### 3.1 Staff Management
1. **View Staff**: Click "Staff" in the sidebar to see the personnel list.
2. **Add Staff**: Click "New Staff", fill in the form (Full Name, Designation, BPS, etc.), and click "Save".
3. **Deactivate Staff**: In the staff list, click "View", then select "Deactivate" (useful for retired or transferred-out personnel).

### 3.2 Leave Registration
1. **Register Leave**: Go to "Leave" → "Add New".
2. **Details**: Select the staff member, leave type, and dates.
3. **Substitute**: Assign another staff member to handle duties during this period.
4. **Approval**: Status defaults to "Active" until completed.

### 3.3 Transfer Tracking
1. **New Transfer**: Go to "Transfers" → "New Order".
2. **States**:
    - **Ordered**: Official order issued.
    - **Relieved**: Personnel has left the current post.
    - **Completed**: Personnel has joined the new post.
3. **Transit**: Staff in the "Relieved" state but not yet "Completed" are flagged as "In Transit".

### 3.4 Revenue Logging (for Postmasters)
1. **Log Daily Revenue**: Navigate to "Revenue" → "New Entry".
2. **Submit**: Enter the revenue figures for the specified date and save.
3. **View Trends**: The Chart on the Dashboard provides a visual overview of revenue progress.

## 4. ASPO Office Specialized Tools

### 4.1 Attendance Register
- **Grid View**: A 31-day grid showing all staff.
- **Marking**: Click a cell to select status: **Present**, **On Duty**, **On Leave**, **Absent**, or **Holiday**.
- **Leave Linking**: For "On Leave", you must select a registered DSPS leave memo.

### 4.2 EDBO Inspection Programme
- **Annual Plan**: View the yearly inspection schedule for all EDBOs assigned to the Overseer.
- **Complete Assignment**: Once an inspection is done, update the status to "Completed" with the actual date and any remarks.

## 5. Account and Settings
- **Profile**: View your account details.
- **Audit Log**: (Admin only) Review all changes made to the system records for accountability.

## 6. Troubleshooting
- **Login Issues**: Ensure caps lock is off. Contact support if the problem persists.
- **Page Not Found**: Re-navigate to the Dashboard to reset the view.
- **Data Not Updating**: Refresh the page or double-check your internet connection.
